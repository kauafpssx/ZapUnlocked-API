---
title: "Imagem do QR"
icon: "image"
api: "GET /qr/image"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna a imagem do QR Code como **binário PNG** (`Content-Type: image/png`). Use para exibir o QR em interfaces próprias via tag `<img>` ou download direto.

<Note>
  Este endpoint retorna imagem binária, não JSON. Para obter o QR Code em Base64, use o endpoint `GET /status/stream`.
</Note>

<ResponseExample>

```json 200 Sucesso
{
  "_note": "Retorna binário PNG — Content-Type: image/png"
}
```

```json 404 QR não disponível
{
  "detail": {
    "error": "QR Code não disponível",
    "message": "WhatsApp já está conectado"
  }
}
```

```json 404 Aguardando
{
  "detail": {
    "error": "QR Code não disponível",
    "message": "Aguardando geração do QR Code..."
  }
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```

</ResponseExample>

---
title: 'Código de Pareamento por Número'
icon: 'mobile-screen'
api: "GET /settings/phone-code/{phone}"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Gera um código de 8 dígitos para conectar um número de WhatsApp sem QR Code, via **Dispositivos vinculados › Conectar com número de telefone**.

<Warning>
  O cliente deve estar aguardando conexão (sem sessão ativa). Se já houver uma sessão, desconecte primeiro via `POST /qr/logout`.
</Warning>

### Path Parameter

<ParamField path="phone" type="string" required>
  Número de telefone com código do país (ex: `5511999999999`).
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "phone": "5511999999999",
  "code": "ABCD-1234"
}
```

```json 408 Timeout
{
  "detail": "Timeout ao gerar código de pareamento"
}
```

```json 503 Não conectado
{
  "detail": "WhatsApp não conectado (cliente deve estar aguardando conexão)"
}
```

</ResponseExample>

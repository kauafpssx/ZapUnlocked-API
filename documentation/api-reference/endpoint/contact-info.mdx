---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Este endpoint permite obter informações detalhadas de um número específico, como foto de perfil, recado (status) e informações comerciais.

### Body

<ParamField body="number" type="string" required>
  O número do contato que você deseja consultar (ex: `5511999999999`).
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem sucedida.
</ResponseField>

<ResponseField name="exists" type="boolean">
  Indica se o número está registrado no WhatsApp.
</ResponseField>

<ResponseField name="profilePictureUrl" type="string">
  URL da foto de perfil do contato (se disponível).
</ResponseField>

<ResponseField name="about" type="string">
  O recado/status atual do usuário (ex: "Disponível").
</ResponseField>

<ResponseField name="businessProfile" type="object">
  Se for uma conta comercial, retorna detalhes como descrição, categoria e email.
</ResponseField>

---
title: "Enviar Localização"
icon: "location-dot"
api: "POST /send_location"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma localização geográfica para um contato ou grupo.

### Corpo da Requisição (Body)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="lat" type="number" required>
  Latitude da localização (ex: `-23.5505`).
</ParamField>

<ParamField body="lng" type="number" required>
  Longitude da localização (ex: `-46.6333`).
</ParamField>

<ParamField body="name" type="string">
  Nome do local exibido na mensagem (ex: `"Av. Paulista, São Paulo"`).
</ParamField>

<ParamField body="address" type="string">
  Endereço completo (ex: `"Av. Paulista, 1578 - Bela Vista, São Paulo - SP"`).
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a localização foi enviada com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva do status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Localização enviada ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```

</ResponseExample>

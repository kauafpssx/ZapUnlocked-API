---
title: "Remover Webhook"
icon: "trash"
api: "DELETE /webhooks/{name}"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Remove permanentemente um webhook pelo nome. O webhook para de receber eventos imediatamente.

### Path

<ParamField path="name" type="string" required>
  Nome do webhook a ser removido (ex: `meu-crm`).
</ParamField>

<Warning>
  A remoção é permanente. Para pausar temporariamente sem perder a configuração, use [toggle](/api-reference/endpoint/webhooks-toggle) com `active: false`.
</Warning>


<ResponseField name="success" type="boolean">
  Indica se o webhook foi removido com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva do status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Webhook 'meu-crm' removido"
}
```

```json 404 Erro
{
  "success": false,
  "message": "Webhook 'meu-crm' não encontrado"
}
```

</ResponseExample>

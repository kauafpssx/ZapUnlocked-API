---
title: "Enviar Arquivo"
icon: "file-import"
api: "POST /send_document"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia qualquer tipo de arquivo (PDF, DOCX, ZIP, etc). Limite de 400MB configurado na API (suporta até 2GB no protocolo).

### Corpo da Requisição (Body)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país.
</ParamField>

<ParamField body="document_url" type="string" required>
  URL pública direta do arquivo.
</ParamField>

<ParamField body="fileName" type="string">
  Nome customizado que o arquivo terá no WhatsApp (ex: `documento_importante.pdf`).
</ParamField>


<ResponseExample>
```json 200 Sucesso
{
  "success": true,
  "message": "Documento enviado com sucesso ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```
</ResponseExample>

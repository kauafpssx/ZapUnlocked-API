---
title: "Listar Eventos"
icon: "bolt"
api: "GET /webhooks/events"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna a lista completa de todos os tipos de eventos disponíveis para inscrição em webhooks.


<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem-sucedida.
</ResponseField>

<ResponseField name="events" type="array">
  Lista de strings com todos os event types disponíveis.
</ResponseField>

<ResponseExample>

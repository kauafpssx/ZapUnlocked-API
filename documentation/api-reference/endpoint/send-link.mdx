---
title: "Enviar Link"
icon: "link"
api: "POST /send_link"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um link com card de preview (Open Graph). O WhatsApp exibe título, descrição e imagem da URL automaticamente.

### Corpo da Requisição (Body)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="url" type="string" required>
  URL completa a ser enviada (ex: `"https://github.com/kauafpssx/ZapUnlocked-API"`).
</ParamField>

<ParamField body="title" type="string">
  Título exibido no preview. Se omitido, usa o `og:title` da página.
</ParamField>

<ParamField body="description" type="string">
  Descrição exibida no preview. Se omitido, usa o `og:description` da página.
</ParamField>

<ParamField body="caption" type="string">
  Texto adicional enviado junto com o link.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se o link foi enviado com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva do status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Link enviado ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```

</ResponseExample>

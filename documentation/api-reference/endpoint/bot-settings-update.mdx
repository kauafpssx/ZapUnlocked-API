---
title: 'Atualizar Config. Bot'
icon: 'sliders'
api: "POST /settings/bot"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Atualiza as configurações do bot. Todos os campos são opcionais — envie apenas os que deseja alterar.

### Body

<ParamField body="ai_tag_enabled" type="boolean">
  Ativa ou desativa a tag de IA nas mensagens.
</ParamField>

<ParamField body="ai_tag_text" type="string">
  Texto da tag de IA (ex: `"[IA]"`, `"🤖"`).
</ParamField>

<ParamField body="ip_control_enabled" type="boolean">
  Ativa ou desativa o controle de acesso por IP.
</ParamField>

<ParamField body="ip_whitelist" type="array">
  Lista de IPs permitidos (somente se `ip_control_enabled: true`).
</ParamField>

<ParamField body="ip_blacklist" type="array">
  Lista de IPs bloqueados.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Configurações do bot atualizadas",
  "settings": {
    "ai_tag_enabled": true,
    "ai_tag_text": "🤖",
    "ip_control_enabled": false,
    "ip_whitelist": [],
    "ip_blacklist": []
  }
}
```

</ResponseExample>

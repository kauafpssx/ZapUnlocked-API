---
title: "Ativar / Desativar"
icon: "toggle-on"
api: "POST /webhooks/{name}/toggle"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Ativa ou desativa um webhook sem remover sua configuração. Webhooks inativos não recebem nenhum evento.

### Path

<ParamField path="name" type="string" required>
  Nome do webhook (ex: `meu-crm`).
</ParamField>

### Corpo da Requisição (Body)

<ParamField body="active" type="boolean" required>
  `true` para ativar, `false` para desativar.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se o estado foi alterado com sucesso.
</ResponseField>

<ResponseField name="webhook" type="object">
  Dados do webhook com o novo estado `active`.
</ResponseField>

<ResponseExample>

```json 200 Desativado
{
  "success": true,
  "webhook": {
    "name": "meu-crm",
    "url": "https://meu-crm.com/hook",
    "events": ["*"],
    "active": false
  }
}
```

```json 404 Erro
{
  "success": false,
  "message": "Webhook 'meu-crm' não encontrado"
}
```

</ResponseExample>

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Define a mensagem enviada automaticamente quando uma chamada é rejeitada.

### Body

<ParamField body="value" type="string" required>
  Texto da mensagem enviada ao chamador após a rejeição. Suporta placeholders: `{{name}}`, `{{phone}}`, `{{day}}`.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "call_reject_message": "No momento não posso atender ligações. Me envie uma mensagem! 😊"
}
```

</ResponseExample>

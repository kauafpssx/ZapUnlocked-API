---
title: "Enviar Contato"
icon: "address-card"
api: "POST /send_contact"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um contato em formato vCard. Use `/send_contact` para um único contato ou `/send_contacts` para múltiplos.

### Corpo da Requisição (Body)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="contactPhone" type="string" required>
  Número do contato a ser enviado (ex: `5511988887777`).
</ParamField>

<ParamField body="contactName" type="string" required>
  Nome exibido no vCard (ex: `"João Silva"`).
</ParamField>

<ParamField body="contactOrg" type="string">
  Organização/empresa do contato.
</ParamField>

<Note>
  **Múltiplos contatos**: Use `POST /send_contacts` com o campo `contacts` sendo uma lista de objetos com `phone`, `name` e opcionalmente `org`. O corpo do destinatário (`phone`) continua obrigatório.
</Note>


<ResponseField name="success" type="boolean">
  Indica se o contato foi enviado com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva do status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Contato enviado ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```

</ResponseExample>

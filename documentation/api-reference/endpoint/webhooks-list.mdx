<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna todos os webhooks cadastrados. Para detalhar um webhook específico, use `GET /webhooks/{name}`.

### GET /webhooks — Listar todos

Retorna um array com todos os webhooks cadastrados, incluindo status e configurações.

### GET /webhooks/{name} — Detalhar um

<ParamField path="name" type="string" required>
  Nome do webhook a ser consultado (ex: `meu-crm`).
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida.
</ResponseField>

<ResponseField name="webhooks" type="array">
  *(Apenas em GET /webhooks)* Lista de todos os webhooks cadastrados.
</ResponseField>

<ResponseField name="webhook" type="object">
  *(Apenas em GET /webhooks/{name})* Dados completos do webhook.
</ResponseField>

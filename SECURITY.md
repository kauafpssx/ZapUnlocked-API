# Security Policy

## Supported Versions

| Version | Security Support |
|---|---|
| latest (main) | Yes |
| old versions | No |

Always use the latest version of the `main` branch.

## Report a Vulnerability

**Never open a public issue to report security vulnerabilities.**

Report privately via **GitHub Security Advisories**:

1. Go to [Security > Advisories](https://github.com/kauafpssx/ZapUnlocked-API/security/advisories/new)
2. Fill in details: description, impact, steps to reproduce, and affected version
3. Await a response within **72 hours**

If you prefer, send an email: **contato@kauafpss.com.br**

## What to Expect

- Confirmation of receipt within 72 hours
- Impact and priority assessment
- Coordinated fix and release with the reporter
- Public credit to the reporter (if desired)

## <Icon icon="key" iconType="sharp-regular" color="#9d4fe6" size={22} /> Autenticação

Todos os endpoints exigem a `API_KEY` definida no `.env`. Dois métodos aceitos:

<AccordionGroup>
  <Accordion title="Header x-api-key (Recomendado)" icon="key">
    ```bash
    x-api-key: sua_chave_secreta_aqui
    ```
  </Accordion>
  <Accordion title="Query parameter API_KEY" icon="link">
    ```bash
    ?API_KEY=sua_chave_secreta_aqui
    ```
    Útil para acessar `/qr` e `/qr/image` diretamente no navegador.
  </Accordion>
</AccordionGroup>

<Warning>
  Nunca exponha sua `API_KEY` no lado do cliente (frontend, app mobile). Qualquer pessoa com a chave tem controle total sobre o WhatsApp conectado. Sempre faça chamadas a partir de um backend.
</Warning>

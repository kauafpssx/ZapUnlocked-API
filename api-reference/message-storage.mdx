A **ZapUnlocked-API** armazena automaticamente as mensagens recebidas para permitir buscas no histórico e análises. Este sistema é otimizado para economizar espaço usando **compressão JSON**.

## <Icon icon="clock" iconType="sharp-regular" color="#9d4fe6" size={22} /> Quando as Mensagens São Salvas

As mensagens **só começam a ser armazenadas quando a API está conectada e em execução**. Mensagens enviadas enquanto o bot está offline não são recuperadas retroativamente.

<Note>
  O armazenamento é incremental: cada nova mensagem recebida é adicionada ao histórico do contato correspondente.
</Note>

---

## <Icon icon="list-ol" iconType="sharp-regular" color="#9d4fe6" size={22} /> Limites de Armazenamento

Para manter o sistema leve e eficiente, aplicamos os seguintes limites:

| Tipo | Limite |
| :--- | :--- |
| **Mensagens por Chat** | Últimas 100 mensagens |
| **Contatos Recentes** | 1.000 contatos interagidos |

<Warning>
  Quando o limite é atingido, as mensagens ou contatos mais antigos são removidos automaticamente para dar espaço aos novos.
</Warning>

---

## <Icon icon="file-zipper" iconType="sharp-regular" color="#9d4fe6" size={22} /> Compressão e Economia de Espaço

Todos os arquivos JSON de mensagens são **comprimidos com gzip** antes de serem salvos no disco. Isso reduz drasticamente o consumo de armazenamento.

<Steps>
  <Step title="Recebimento" icon="inbox">
    Mensagem chega pelo WhatsApp e é capturada pelo handler.
  </Step>
  <Step title="Indexação" icon="list">
    Os dados são estruturados como JSON com metadados do contato e conteúdo.
  </Step>
  <Step title="Compressão" icon="file-zipper">
    O JSON é comprimido com gzip antes de qualquer escrita no disco.
  </Step>
  <Step title="Armazenamento" icon="floppy-disk">
    O arquivo comprimido é salvo no disco, ocupando até 70% menos espaço.
  </Step>
  <Step title="Leitura" icon="magnifying-glass">
    Ao consultar, o arquivo é descomprimido em memória, processado e retornado via API.
  </Step>
</Steps>

## <Icon icon="calculator" iconType="sharp-regular" color="#9d4fe6" size={22} /> Consumo Aproximado de Armazenamento

Com base em testes reais, cada mensagem ocupa aproximadamente **0,4 KB** após a compressão. Alguns exemplos:

| Número de Mensagens | Espaço Usado |
| :--- | :--- |
| 10 mensagens (1 chat) | ~1 KB |
| 100 mensagens (1 chat completo) | ~8 KB |
| 1.000 mensagens (10 chats) | ~80 KB |
| 10.000 mensagens (100 chats) | ~800 KB |
| 100.000 mensagens (limite máximo) | ~80 MB |

<Tip>
  Com o limite de **100 mensagens por chat** e **1.000 contatos**, o máximo teórico é de aproximadamente **80 MB** de armazenamento de mensagens.
</Tip>

---

## <Icon icon="broom" iconType="sharp-regular" color="#9d4fe6" size={22} /> Limpeza de Dados

Para liberar espaço ou resetar o histórico, use o endpoint:  
[`DELETE /management/cleanup`](/api-reference/endpoint/cleanup).

<Warning>
  Esta ação é **irreversível** e deleta todos os JSONs de mensagens e índices de contatos. A sessão do WhatsApp não é desconectada.
</Warning>

---
title: "Lista de Botões"
icon: "list"
api: "POST /messages/send-button-list"
description: "Envia uma mensagem com botões de resposta rápida."
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma mensagem com botões de tipos mistos (URL, ligação, OTP, resposta rápida) em um único array. Aparecem como chips abaixo da mensagem. Quando o usuário clica, o evento `message.button_reply` é disparado para o(s) webhook(s) cadastrado(s) inscrito(s) nesse evento.

<Note>
  ⚠ Botão PIX **não** é suportado em listas combinadas — use `/messages/send-button-pix` separadamente.
</Note>

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="message" type="string" required>
  Texto principal exibido acima dos botões.
</ParamField>

<ParamField body="title" type="string" required>
  Título da mensagem exibido no topo.
</ParamField>

<ParamField body="text" type="string" required>
  Texto descritivo abaixo do título.
</ParamField>

<ParamField body="footer" type="string">
  Texto de rodapé exibido abaixo dos botões.
</ParamField>

<ParamField body="buttons" type="array" required>
  Lista de botões (máx 5). Tipos podem ser misturados:

  <Accordion title="Botão quick_reply">
    <ParamField body="type" type="string" required>
      `"quick_reply"` — gera evento `message.button_reply` ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="id" type="string">
      ID retornado no evento (gerado automaticamente se omitido).
    </ParamField>
  </Accordion>

  <Accordion title="Botão url">
    <ParamField body="type" type="string" required>
      `"url"` — abre uma URL ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="url" type="string" required>
      URL a abrir (ex: `"https://zapunlocked.com"`).
    </ParamField>
  </Accordion>

  <Accordion title="Botão call">
    <ParamField body="type" type="string" required>
      `"call"` — inicia uma ligação ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="callPhone" type="string" required>
      Número a ligar (ex: `"+5511999999999"`).
    </ParamField>
  </Accordion>

  <Accordion title="Botão otp">
    <ParamField body="type" type="string" required>
      `"otp"` — copia um código ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="code" type="string" required>
      Código a ser copiado.
    </ParamField>
  </Accordion>
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como o campo `reply` é interpretado: `id` (padrão) ou `text`.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>

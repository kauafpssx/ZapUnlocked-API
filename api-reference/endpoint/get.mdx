Ao acessar a URL raiz da sua instância no navegador, a API exibe uma **página interativa de boas-vindas** com tudo que você precisa para iniciar a conexão.

<Note>
  Rota pública - não exige autenticação. Não retorna JSON.
</Note>

---

## <Icon icon="link" iconType="sharp-regular" color="#9d4fe6" size={22} /> Como acessar

Abra no navegador a URL da sua instância:

<CodeGroup>
```bash Local icon="house"
http://localhost:8300/
```
```bash VPS icon="server"
http://SEU_IP:8300/
```
```bash Alwaysdata icon="cloud"
http://services-[usuario].alwaysdata.net:8300/
```
```bash Domínio próprio icon="globe"
https://minha-api.exemplo.com/
```
</CodeGroup>

---

## <Icon icon="eye" iconType="sharp-regular" color="#9d4fe6" size={22} /> O que aparece

A página exibe duas abas:

<AccordionGroup>
  <Accordion title="Via URL" icon="link">
    Mostra o link completo para acessar o painel do QR Code diretamente no navegador:

    ```
    http://SUA_URL:8300/qr?API_KEY=SUA_SENHA_SECRETA
    ```

    Clique ou cole no navegador para escanear o QR Code e conectar o WhatsApp.
  </Accordion>
  <Accordion title="Via Request" icon="code">
    Mostra o endpoint para obter o QR Code como imagem PNG via requisição:

    ```
    GET http://SUA_URL:8300/qr/image?API_KEY=SUA_SENHA_SECRETA
    ```

    Útil para integrar o QR Code em painéis ou sistemas próprios.
  </Accordion>
</AccordionGroup>

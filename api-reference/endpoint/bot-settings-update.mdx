<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Atualiza as configurações do bot. Todos os campos são opcionais — envie apenas os que deseja alterar.

### Corpo da Requisição

<ParamField body="ai_tag_enabled" type="boolean">
  Ativar ou desativar a tag de IA nas mensagens.
</ParamField>

<ParamField body="ai_tag_text" type="string">
  Texto da tag de IA (ex: `"[IA]"`, `"🤖"`).
</ParamField>

<ParamField body="ip_control_enabled" type="boolean">
  Ativar ou desativar o controle de acesso por IP. Para gerenciar as listas, use os endpoints de [Regras de IP](/api-reference/endpoint/ip-rules-list).
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Bot settings updated.",
  "settings": {
    "ai_tag_enabled": true,
    "ai_tag_text": "🤖",
    "ip_control_enabled": false
  }
}
```

</ResponseExample>

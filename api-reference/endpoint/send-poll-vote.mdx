---
title: "Votar em Enquete"
icon: "square-check"
api: "POST /messages/send-poll-vote"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Registra um voto em uma enquete existente. Útil para simular participação do bot em enquetes ou para testes.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Telefone do chat onde a enquete foi enviada (ex: `5511999999999`).
</ParamField>

<ParamField body="pollId" type="string" required>
  ID da mensagem da enquete.
</ParamField>

<ParamField body="options" type="array" required>
  Lista dos textos das opções selecionadas:
  ```json
  ["Azul"]
  ```
  Para remover voto, envie array vazio: `[]`.
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se o voto foi registrado com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Poll vote sent."
}
```

```json 404 Não encontrado
{
  "success": false,
  "message": "Poll not found."
}
```

</ResponseExample>

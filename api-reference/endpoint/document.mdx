---
title: "Enviar Arquivo"
icon: "file-import"
api: "POST /send_document"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia qualquer tipo de arquivo (PDF, DOCX, ZIP, etc.). A API está configurada para 400 MB; o protocolo suporta até 2 GB.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país.
</ParamField>

<ParamField body="document_url" type="string" required>
  URL pública do arquivo.
</ParamField>

<ParamField body="fileName" type="string">
  Nome personalizado exibido no WhatsApp (ex: `documento_importante.pdf`).
</ParamField>

<ParamField body="caption" type="string">
  Legenda exibida junto ao arquivo.
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como o campo `reply` é interpretado: `id` (padrão) ou `text`.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>


<ResponseExample>
```json 200 Sucesso
{
  "success": true,
  "message": "Document sent successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```
</ResponseExample>

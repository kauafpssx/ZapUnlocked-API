<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Ativa ou desativa o controle de IP (whitelist/blacklist) nas requisições à API.

### Corpo da Requisição

<ParamField body="enabled" type="boolean" required>
  Define se o controle de IP está ativo (`true`) ou inativo (`false`).
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "IP control updated."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

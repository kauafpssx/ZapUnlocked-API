---
title: "Deletar Mensagem"
icon: "trash"
api: "POST /messages/delete"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Deleta uma mensagem para todos os participantes do chat (equivalente ao "apagar para todos" do WhatsApp).

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário/chat com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="messageId" type="string" required>
  ID da mensagem a deletar (com `type: "id"`), ou texto exato (com `type: "text"`).
</ParamField>

<ParamField body="fromMe" type="boolean" default="true">
  Se `true`, deleta a mensagem enviada pelo bot. Use `false` para deletar mensagem recebida (apenas local).
</ParamField>

<ParamField body="type" type="string" default="id">
  Como localizar a mensagem: `"id"` (pelo ID) ou `"text"` (pelo conteúdo).
</ParamField>

<Warning>
  A exclusão é permanente e afeta todos os participantes. Mensagens muito antigas podem não ser deletáveis dependendo do cliente WhatsApp do destinatário.
</Warning>


<ResponseField name="success" type="boolean">
  Indica se a mensagem foi deletada com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Message deleted."
}
```

```json 404 Não encontrado
{
  "success": false,
  "message": "Message not found."
}
```

</ResponseExample>

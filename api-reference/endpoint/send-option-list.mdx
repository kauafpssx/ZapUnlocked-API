---
title: "Lista de Opções"
icon: "list-ul"
api: "POST /messages/send-option-list"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma lista de opções interativa. O usuário toca no botão, expande a lista e seleciona uma opção. A seleção dispara o evento `message.list_reply` para o(s) webhook(s) cadastrado(s) inscrito(s) nesse evento.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="title" type="string" required>
  Título exibido no topo da lista.
</ParamField>

<ParamField body="text" type="string" required>
  Texto da mensagem exibido acima do botão de lista.
</ParamField>

<ParamField body="options" type="array" required>
  Lista de opções disponíveis:

  <Accordion title="Estrutura da opção">
    <ParamField body="title" type="string" required>
      Texto principal do item.
    </ParamField>
    <ParamField body="description" type="string">
      Descrição secundária do item (ex: preço, detalhe).
    </ParamField>
  </Accordion>
</ParamField>

<ParamField body="button_text" type="string" default="See options">
  Texto exibido no botão que abre a lista.
</ParamField>

<ParamField body="footer" type="string">
  Texto de rodapé.
</ParamField>

<ParamField body="description" type="string">
  Texto de descrição abaixo do título.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a lista foi enviada com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Interactive message sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: 'Enviar Sticker'
icon: 'note-sticky'
api: "POST /send_sticker"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um sticker. Aceita URL pública ou upload direto de imagem via `multipart/form-data`.

### Corpo da Requisição (JSON ou multipart/form-data)

<ParamField body="phone" type="string" required>
  Número do contato com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="url" type="string">
  URL pública da imagem. Se omitido, use `file` para upload direto.
</ParamField>

<ParamField body="file" type="file">
  Arquivo de imagem via multipart/form-data. Alternativa a `url`.
</ParamField>

<ParamField body="pack" type="string">
  Nome do pacote de stickers (ex: "Meu Pacote").
</ParamField>

<ParamField body="author" type="string">
  Nome do autor do sticker (ex: "Bot").
</ParamField>

<ParamField body="resize_mode" type="string" default="pad">
  Modo de redimensionamento da imagem.
  
  | Modo | O que faz | Parâmetro extra |
  | :--- | :--- | :--- |
  | `pad` | Adiciona bordas para preencher (padrão). | `pad_color` |
  | `crop` | Corta a imagem para preencher o espaço. | - |
  | `stretch` | Estica a imagem (pode distorcer). | - |
  | `blur` | Preenche o fundo com versão desfocada. | `blur_intensity` |
  | `transparent` | Redimensiona mantendo proporção com fundo transparente. | - |
</ParamField>

<ParamField body="pad_color" type="string" default="black">
  Cor de fundo (somente para `pad`). Ex: `#FFFFFF` ou `transparent`.
</ParamField>

<ParamField body="blur_intensity" type="number" default="20">
  Intensidade do desfoque (somente para `blur`). 1 a 100.
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>


<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Sticker sent successfully."
}
```

```json 500 Erro interno
{
  "error": "Erro ao converter imagem"
}
```

</ResponseExample>

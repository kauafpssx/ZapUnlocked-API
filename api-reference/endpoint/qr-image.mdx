---
title: "QR Code"
icon: "image"
api: "GET /qr/image"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna a imagem do QR Code como **PNG binário** (`Content-Type: image/png`). Use para exibir o QR em interfaces personalizadas via tag `<img>` ou download direto.

<Note>
  Este endpoint retorna dados de imagem binária, não JSON. Para obter o QR Code em Base64, use `GET /status/stream`.
</Note>

<ResponseExample>

```json 200 Sucesso
{
  "_note": "Retorna PNG binário — Content-Type: image/png"
}
```

```json 404 Já conectado
{
  "detail": {
    "error": "NOT_FOUND",
    "message": "WhatsApp is already connected."
  }
}
```

```json 404 Aguardando
{
  "detail": {
    "error": "NOT_FOUND",
    "message": "Waiting for QR code generation..."
  }
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

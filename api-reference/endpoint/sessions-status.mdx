api: "GET /sessions/{session_id}/status"

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

<ParamField header="session_id" type="string" required>
  ID da sessão a ser consultada.
</ParamField>

Retorna o status atual de conexão da sessão WhatsApp.

### Resposta

<ResponseField name="status" type="string">
  Status atual: `connected`, `disconnected`, `connecting`, `qr_read`.
</ResponseField>

<ResponseField name="connected" type="boolean">
  Indica se a sessão está conectada.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "status": "connected",
  "connected": true
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: "Editar Mensagem"
icon: "pen-to-square"
api: "POST /messages/edit"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Edita o texto de uma mensagem **enviada pelo bot**. A mensagem aparece com a tag "Editado" no WhatsApp de todos os participantes.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do chat com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="messageId" type="string" required>
  ID da mensagem original a editar (com `type: "id"`), ou texto exato (com `type: "text"`).
</ParamField>

<ParamField body="message" type="string" required>
  Novo conteúdo de texto da mensagem.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como localizar a mensagem: `"id"` (pelo ID) ou `"text"` (pelo conteúdo).
</ParamField>

<Warning>
  Apenas mensagens **enviadas pelo próprio bot** podem ser editadas. Tentativas de editar mensagens de terceiros serão ignoradas pelo WhatsApp.
</Warning>


<ResponseField name="success" type="boolean">
  Indica se a mensagem foi editada com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Message edited."
}
```

```json 404 Não encontrado
{
  "success": false,
  "message": "Message not found."
}
```

</ResponseExample>

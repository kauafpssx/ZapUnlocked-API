---
title: "Add Whitelist"
icon: "shield-check"
api: "POST /settings/ip-rules/whitelist"
description: "Adiciona um endereço IP à whitelist."
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Adiciona um endereço IP à **whitelist**. IPs na whitelist sempre têm acesso quando o IP Control está ativo.

### Corpo da Requisição

<ParamField body="ip" type="string" required>
  Endereço IP a adicionar (IPv4 ou IPv6). Ex: `"192.168.1.100"`.
</ParamField>

<ResponseField name="success" type="boolean">
  `true` se o IP foi adicionado com sucesso.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "rules": {
    "whitelist": ["192.168.1.100"],
    "blacklist": []
  }
}
```

```json 400 Campo ausente
{
  "error": "MISSING_FIELD",
  "message": "'ip' is required."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

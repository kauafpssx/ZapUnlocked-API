<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um botão de **pagamento PIX**. O WhatsApp exibe o valor e a chave PIX com um botão para iniciar o pagamento diretamente no app.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="pixKey" type="string" required>
  Chave PIX de destino (CPF, e-mail, telefone, chave aleatória ou CNPJ).
</ParamField>

<ParamField body="pixType" type="string">
  Tipo da chave PIX. Valores aceitos: `CPF`, `CNPJ`, `EMAIL`, `PHONE`, `EVP`.
</ParamField>

<ParamField body="merchantName" type="string">
  Nome do beneficiário exibido na tela de pagamento.
</ParamField>

<ParamField body="message" type="string">
  Texto da mensagem exibido acima do botão.
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como o campo `reply` é interpretado: `id` (padrão) ou `text`.
</ParamField>

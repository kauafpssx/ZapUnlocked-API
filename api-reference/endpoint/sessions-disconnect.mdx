---
title: "Desconectar"
icon: "power-off"
api: "POST /sessions/{session_id}/disconnect"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

<ParamField path="session_id" type="string" required>
  ID da sessão a ser desconectada.
</ParamField>

Desconecta a sessão do WhatsApp. A sessão permanece registrada e pode ser reconectada depois.

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Session '1' disconnected."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: "Criar Webhook"
icon: "plus"
api: "POST /webhooks"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Cria um novo webhook com nome único. O nome é usado como identificador em todas as operações subsequentes.

### Corpo da Requisição

<ParamField body="name" type="string" required>
  Identificador único do webhook. Aceita letras, números, hífens e underscores (ex: `"meu-crm"`, `"dashboard_v2"`).
</ParamField>

<ParamField body="url" type="string" required>
  URL HTTPS que receberá os eventos (ex: `"https://meu-sistema.com/webhook"`).
</ParamField>

<ParamField body="method" type="string" default="POST">
  Método HTTP: `"POST"`, `"GET"` ou `"PUT"`.
</ParamField>

<ParamField body="headers" type="object" default="{}">
  Headers personalizados enviados em cada disparo. Suporta placeholders.
  ```json
  { "Authorization": "Bearer meu-token" }
  ```
</ParamField>

<ParamField body="body" type="object" default="{}">
  JSON personalizado enviado no corpo da requisição. Se vazio, usa o envelope padrão `{event, timestamp, data}`. Suporta placeholders.
</ParamField>

<ParamField body="events" type="array" default='["*"]'>
  Tipos de eventos que disparam este webhook. Use `["*"]` para todos os eventos, ou liste tipos específicos:
  ```json
  ["message.text", "message.button_reply", "message.poll_vote"]
  ```
  Consulte a [lista completa de eventos](/api-reference/endpoint/webhooks-events).
</ParamField>

<ParamField body="active" type="boolean" default="true">
  Se o webhook deve iniciar ativo.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se o webhook foi criado com sucesso.
</ResponseField>

<ResponseField name="webhook" type="object">
  Dados do webhook criado.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "webhook": {
    "name": "meu-crm",
    "url": "https://meu-crm.com/hook",
    "method": "POST",
    "headers": {},
    "body": {},
    "events": ["*"],
    "active": true,
    "created_at": "2025-01-01T12:00:00Z"
  }
}
```

```json 409 Conflito
{
  "detail": {
    "error": "INVALID_FIELD",
    "message": "Webhook 'meu-crm' already exists. Use PUT to update."
  }
}
```

</ResponseExample>

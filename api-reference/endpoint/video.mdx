---
title: "Enviar Vídeo"
icon: "video"
api: "POST /send_video"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um vídeo. Aceita URL pública ou upload direto via `multipart/form-data`. Arquivos < 15 MB são enviados como vídeo normal. Maiores viram documentos (qualidade integral).

### Corpo da Requisição (JSON ou multipart/form-data)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país.
</ParamField>

<ParamField body="url" type="string">
  URL pública do vídeo. Se omitido, use `file` para upload direto.
</ParamField>

<ParamField body="file" type="file">
  Arquivo de vídeo via multipart/form-data. Alternativa a `url`.
</ParamField>

<ParamField body="caption" type="string">
  Legenda opcional para o vídeo.
</ParamField>

<ParamField body="as_document" type="boolean" default="false">
  Força o envio como documento (até 2 GB).
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>


<ResponseExample>
```json 200 Sucesso
{
  "success": true,
  "message": "Video sent successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```
</ResponseExample>

---
title: "Enviar Vídeo"
icon: "video"
api: "POST /send_video"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um vídeo. Arquivos com menos de 15 MB são enviados como vídeo normal. Arquivos maiores são enviados como documentos (qualidade integral).

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país.
</ParamField>

<ParamField body="video_url" type="string" required>
  URL pública do arquivo de vídeo (mp4).
</ParamField>

<ParamField body="caption" type="string">
  Legenda opcional para o vídeo.
</ParamField>

<ParamField body="gifPlayback" type="boolean" default="false">
  Se `true`, envia o vídeo como GIF (em loop, sem som).
</ParamField>

<ParamField body="ptv" type="boolean" default="false">
  Se `true`, envia como vídeo redondo (clipe curto).
</ParamField>

<ParamField body="asDocument" type="boolean" default="false">
  Força o envio como documento (até 2 GB).
</ParamField>


<ResponseExample>
```json 200 Sucesso
{
  "success": true,
  "message": "Video sent successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```
</ResponseExample>

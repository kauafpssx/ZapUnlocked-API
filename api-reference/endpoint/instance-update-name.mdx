<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Renomeia a instância: atualiza o `pushname` no WhatsApp e persiste o nome em `db_config.json`.

### Body

<ParamField body="name" type="string" required>
  Novo nome da instância / pushname do WhatsApp.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "status": "success",
  "message": "Instância renomeada para: Meu Bot v2"
}
```

```json 503 Não conectado
{
  "detail": "WhatsApp não conectado"
}
```

</ResponseExample>

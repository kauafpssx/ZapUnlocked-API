title: 'Renomear Instância'
icon: 'pen-to-square'
api: "PUT /instance/update-name"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Renomeia a instância: atualiza o `pushname` no WhatsApp e persiste o nome em `db_config.json`.

### Corpo da Requisição

<ParamField body="name" type="string" required>
  Novo nome da instância / pushname do WhatsApp.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Instance renamed to: My Bot v2"
}
```

```json 503 Não conectado
{
  "detail": "WhatsApp is not connected."
}
```

</ResponseExample>

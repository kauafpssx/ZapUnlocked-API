---
title: 'Cód. Pareamento'
icon: 'mobile'
api: "POST /qr/pair"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Gera um código de 8 dígitos para conectar ao WhatsApp sem precisar escanear QR Code (útil para emuladores ou quando a câmera não funciona).

### Body

<ParamField body="phone" type="string" required>
  O número de telefone que será conectado, incluindo código do país (ex: `5511999999999`).
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "code": "ABCD-1234"
}
```

```json 400 Falha ao gerar código
{
  "detail": "O código de pareamento não foi capturado. Verifique se o número está formatado corretamente (ex: 5551988887777)"
}
```

```json 409 Já conectado
{
  "detail": {
    "error": "WhatsApp já está conectado. Faça logout antes de parear novamente."
  }
}
```

```json 503 Serviço indisponível
{
  "detail": {
    "error": "Serviço WhatsApp não inicializado"
  }
}
```

</ResponseExample>

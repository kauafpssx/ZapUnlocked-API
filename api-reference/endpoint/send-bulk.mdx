---
title: "Envio em Massa"
icon: "users"
api: "POST /send_bulk"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia mensagens em massa para múltiplos números. O sistema processa a lista sequencialmente, com delay opcional entre cada envio.

### Corpo da Requisição

<ParamField body="phones" type="array" required>
  Lista de números de destinatários com código do país (ex: `["5511999999999", "5511888888888"]`).
</ParamField>

<ParamField body="message" type="string" required>
  Texto da mensagem a ser enviada para todos os números.
</ParamField>

<ParamField body="delay_between" type="string">
  Delay entre cada envio no formato cron ou em milissegundos. Ex: `"1000"` para 1 segundo.
</ParamField>

<ParamField body="delay_message" type="string">
  Delay em segundos antes de enviar cada mensagem individual.
</ParamField>

<ParamField body="delay_typing" type="string">
  Simula digitação por este período (em segundos) antes de enviar cada mensagem.
</ParamField>

<ParamField body="mentioned" type="array">
  Lista de números a serem mencionados na mensagem.
</ParamField>

```json 200 Sucesso
{
  "success": true,
  "message": "Bulk messages sent."
```

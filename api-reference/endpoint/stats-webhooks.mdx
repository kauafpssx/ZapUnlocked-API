---
title: "Stats Webhooks"
icon: "chart-line"
api: "GET /stats/webhooks"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna estatísticas detalhadas dos webhooks, incluindo total de chamadas, falhas e taxa de sucesso.

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "webhooks": {
    "total_calls": 500,
    "success": 480,
    "failures": 20,
    "success_rate": 96.0
  }
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

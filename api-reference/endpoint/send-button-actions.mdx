---
title: "Botões de Ação"
icon: "bolt"
api: "POST /messages/send-button-list"
description: "Envia uma mensagem com botões de URL, ligação ou resposta rápida."
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia botões do tipo **ação**: URL (abre link), ligação (inicia chamada) ou resposta rápida. Podem ser misturados na mesma mensagem.

Quando o usuário clica em um botão de **resposta rápida**, o evento `message.button_reply` é disparado para o(s) webhook(s) cadastrado(s) inscrito(s) nesse evento.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="title" type="string">
  Título exibido no topo da mensagem.
</ParamField>

<ParamField body="message" type="string">
  Texto principal exibido acima dos botões.
</ParamField>

<ParamField body="footer" type="string">
  Texto de rodapé exibido abaixo dos botões (ex: `"ZapUnlocked API"`).
</ParamField>

<ParamField body="buttons" type="array" required>
  Lista de botões de ação. Cada item deve conter `type` e `buttonText`.

  <Accordion title="Botão tipo url">
    <ParamField body="type" type="string" required>
      `"url"` — abre uma URL ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="url" type="string" required>
      URL a abrir (ex: `"https://zapunlocked.com"`).
    </ParamField>
  </Accordion>

  <Accordion title="Botão tipo call">
    <ParamField body="type" type="string" required>
      `"call"` — inicia uma ligação ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="phoneNumber" type="string" required>
      Número a ligar (ex: `"+5511999999999"`).
    </ParamField>
  </Accordion>

  <Accordion title="Botão tipo quick_reply">
    <ParamField body="type" type="string" required>
      `"quick_reply"` — gera evento `message.button_reply` ao clicar.
    </ParamField>
    <ParamField body="buttonText" type="string" required>
      Texto exibido no botão.
    </ParamField>
    <ParamField body="id" type="string">
      ID retornado no evento de webhook (gerado automaticamente se omitido).
    </ParamField>
  </Accordion>
</ParamField>

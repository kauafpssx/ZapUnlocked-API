Verifica se o servidor está pronto para receber requisições. Verifica dependências como banco de dados e conexão WhatsApp.

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "ready": true,
  "dependencies": {
    "database": "ok",
    "whatsapp": "connected"
  }
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

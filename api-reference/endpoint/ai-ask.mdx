---
title: "Perguntar ao Meta AI"
icon: "robot"
api: "POST /ai/ask"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma pergunta para o Meta AI integrado ao WhatsApp e retorna a resposta gerada.

### Corpo da Requisição

<ParamField body="message" type="string" required>
  Texto da pergunta a ser enviada ao Meta AI.
</ParamField>

<ParamField body="timeout" type="number" default="30">
  Tempo máximo em segundos para aguardar a resposta da IA.
</ParamField>

### Resposta

<ResponseField name="text" type="string">
  Texto da resposta gerada pelo Meta AI.
</ResponseField>

<ResponseField name="hasImage" type="boolean">
  Indica se a resposta contém uma imagem gerada.
</ResponseField>

<ResponseField name="messageId" type="string">
  ID da mensagem de resposta no WhatsApp.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "text": "A resposta do Meta AI...",
  "hasImage": false,
  "messageId": "true_5511999999999@c.us_ABCD1234"
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

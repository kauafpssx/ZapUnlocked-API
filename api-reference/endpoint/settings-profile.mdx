<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.

Atualiza as informações de perfil do WhatsApp, como nome e descrição (about).

### Corpo da Requisição

<ParamField body="name" type="string" required>
  Novo nome de exibição do perfil.
</ParamField>

<ParamField body="about" type="string" required>
  Nova descrição (recado) do perfil.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Profile updated successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: 'Editar Perfil'
icon: 'user-gear'
api: "POST /settings/profile"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Atualiza o nome de exibição e a foto de perfil do bot.

### Corpo da Requisição

<ParamField body="name" type="string">
  Novo nome de exibição do perfil do bot.
</ParamField>

<ParamField body="newProfilePictureUrl" type="string">
  URL pública de uma imagem (JPEG/PNG) para definir como nova foto de perfil.
</ParamField>

<Note>
  É necessário enviar pelo menos um campo: `name` ou `newProfilePictureUrl`.
</Note>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "updated": [
    "Name changed to: My Bot",
    "Profile photo updated (PictureID: abc123)"
  ]
}
```

```json 400 Campos ausentes
{
  "error": "MISSING_FIELD",
  "message": "Provide 'name' or 'newProfilePictureUrl' to update (at least one required)."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: "Listar Eventos"
icon: "bolt"
api: "GET /webhooks/events"
description: "Retorna a lista completa dos 40 tipos de eventos disponíveis."
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Retorna todos os tipos de evento que podem ser usados no campo `events` ao criar ou atualizar um webhook. Use `["*"]` para inscrever em todos de uma vez.

<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem-sucedida.
</ResponseField>

<ResponseField name="events" type="array">
  Lista de todos os 40 tipos de evento disponíveis.
</ResponseField>

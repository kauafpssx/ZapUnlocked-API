---
title: "Enviar Link"
icon: "link"
api: "POST /send_link"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um link com cartão de pré-visualização (Open Graph). O WhatsApp exibe automaticamente o título, descrição e imagem.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="url" type="string" required>
  URL completa a ser enviada (ex: `"https://github.com/kauafpssx/ZapUnlocked-API"`).
</ParamField>

<ParamField body="title" type="string">
  Título exibido na pré-visualização. Se omitido, usa o `og:title` da página.
</ParamField>

<ParamField body="description" type="string">
  Descrição exibida na pré-visualização. Se omitida, usa o `og:description` da página.
</ParamField>

<ParamField body="caption" type="string">
  Texto adicional enviado junto com o link.
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se o link foi enviado com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Link sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: "Enviar Contato"
icon: "address-card"
api: "POST /send_contact"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um contato como vCard. Use `/send_contact` para um único contato ou `/send_contacts` para múltiplos.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="contactName" type="string" required>
  Nome exibido no vCard (ex: `"João da Silva"`).
</ParamField>

<ParamField body="contactPhone" type="string" required>
  Número de telefone do contato a enviar (ex: `5511988887777`).
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como o campo `reply` é interpretado: `id` (padrão) ou `text`.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>

<Note>
  **Múltiplos contatos**: Use `POST /send_contacts` com o campo `contacts` como lista de objetos com `phone` e `name`. O campo `phone` do destinatário ainda é obrigatório.
</Note>


<ResponseField name="success" type="boolean">
  Indica se o contato foi enviado com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Contact sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

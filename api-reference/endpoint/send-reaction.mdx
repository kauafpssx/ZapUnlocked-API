---
title: 'Enviar Reação'
icon: 'face-smile'
api: "POST /send_reaction"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Reage a uma mensagem específica com um emoji.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do contato com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="messageId" type="string" required>
  ID da mensagem a reagir (com `type: "id"`), ou texto exato (com `type: "text"`).
</ParamField>

<ParamField body="emoji" type="string" required>
  Emoji de reação (ex: `"👍"`). Envie string vazia `""` para remover a reação.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como localizar a mensagem: `"id"` (pelo ID, padrão) ou `"text"` (pelo conteúdo).
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
    "success": true,
    "message": "Reaction sent."
}
```

</ResponseExample>

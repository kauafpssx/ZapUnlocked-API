---
title: "Enviar Enquete"
icon: "chart-bar"
api: "POST /messages/send-poll"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma enquete nativa do WhatsApp com múltiplas opções. Os participantes votam diretamente no app e os votos disparam eventos de webhook.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="name" type="string" required>
  Pergunta/título da enquete (ex: `"Qual sua cor favorita?"`).
</ParamField>

<ParamField body="options" type="array" required>
  Lista de opções. Mínimo 2, máximo 12 opções.
  ```json
  ["Azul", "Verde", "Vermelho", "Amarelo"]
  ```
</ParamField>

<ParamField body="selectableCount" type="integer" default="1">
  Máximo de opções que o participante pode selecionar. `1` = escolha única, `0` = ilimitado.
</ParamField>

<Note>
  Os votos são recebidos via webhook com o evento `message.poll_vote`. Configure um webhook para processar os resultados em tempo real.
</Note>


<ResponseField name="success" type="boolean">
  Indica se a enquete foi enviada com sucesso.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Poll sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

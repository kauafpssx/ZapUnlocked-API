---
title: "Criar Sessão"
icon: "plus"
api: "POST /sessions"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Cria uma nova sessão WhatsApp. Útil para gerenciar múltiplas contas simultaneamente.

### Corpo da Requisição

<ParamField body="name" type="string">
  Nome da nova sessão. Se omitido, um nome padrão será gerado.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "session": {
    "id": "2",
    "name": "Nova Sessão",
    "created_at": "2026-06-08T23:19:00Z",
    "active": true
  }
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

---
title: "Deletar Sessão"
icon: "trash"
api: "DELETE /sessions/{session_id}"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

<ParamField header="session_id" type="string" required>
  ID da sessão a ser deletada.
</ParamField>

Remove permanentemente uma sessão WhatsApp e seus dados associados.

<Warning>
  Esta ação é irreversível. Todos os dados da sessão serão perdidos.
</Warning>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Session deleted."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

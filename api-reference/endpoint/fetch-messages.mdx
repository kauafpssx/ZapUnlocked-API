<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Busca mensagens armazenadas em disco (JSON GZIP) com filtros avançados.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do telefone para buscar o histórico (ex: `5511999999999`).
</ParamField>

<ParamField body="limit" type="integer" default="20">
  Número máximo de mensagens a retornar.
</ParamField>

<ParamField body="type" type="string" default="all">
  Filtra por tipo de mídia: `"all"`, `"text"`, `"image"`, `"video"`, `"audio"`, `"document"`, etc.
</ParamField>

<ParamField body="query" type="string">
  Texto para buscar dentro das mensagens (filtro de conteúdo).
</ParamField>

<ParamField body="onlyReactions" type="boolean" default="false">
  Se `true`, retorna apenas mensagens que receberam reações.
</ParamField>

<ParamField body="reactionEmoji" type="string">
  Filtrar mensagens reagidas com um emoji específico (ex: "👍").
</ParamField>

<ParamField body="webhook" type="object">
  Configuração de webhook para disparar com o resultado da busca. Suporta placeholders como `{{requested}}` e `{{found}}`.
</ParamField>

api: "POST /management/fetch_messages"

<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Busca mensagens salvas no disco (JSON GZIP) com filtros avançados.

### Body

<ParamField body="phone" type="string" required>
  Número do telefone para buscar o histórico (ex: `5511999999999`).
</ParamField>

<ParamField body="limit" type="integer" default="50">
  Quantidade máxima de mensagens a retornar.
</ParamField>

<ParamField body="type" type="string" default="all">
  Tipo de mensagem: `sent` (enviadas), `received` (recebidas) ou `all` (todas).
</ParamField>

<ParamField body="query" type="string">
  Texto para buscar dentro das mensagens (filtro por conteúdo).
</ParamField>

<ParamField body="onlyReactions" type="boolean" default="false">
  Se `true`, retorna apenas mensagens que receberam reações.
</ParamField>

<ParamField body="reactionEmoji" type="string">
  Filtra mensagens reagidas com um emoji específico (ex: "👍").
</ParamField>

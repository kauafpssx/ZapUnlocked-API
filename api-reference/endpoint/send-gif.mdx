---
title: "Enviar GIF"
icon: "file-gif"
api: "POST /send_gif"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia um GIF animado. Aceita URL pública ou upload direto via `multipart/form-data`.

### Corpo da Requisição (JSON ou multipart/form-data)

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="url" type="string">
  URL pública do GIF. Se omitido, use `file` para upload direto.
</ParamField>

<ParamField body="file" type="file">
  Arquivo GIF via multipart/form-data. Alternativa a `url`.
</ParamField>

<ParamField body="caption" type="string">
  Legenda exibida abaixo do GIF.
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "GIF sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

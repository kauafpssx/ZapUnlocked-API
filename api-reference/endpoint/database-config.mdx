---
title: "Config. Limpeza"
icon: "clock-rotate-left"
api: "POST /management/database/config"
description: "Define o intervalo de tempo para a limpeza automática do banco de dados."
---

### Corpo da Requisição

<ParamField body="interval" type="number" placeholder="1440" required>
  Intervalo em minutos entre cada limpeza automática. O padrão é 1440 (24
  horas).
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a configuração foi salva.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Intervalo atualizado para 60 minutos"
}
```

</ResponseExample>

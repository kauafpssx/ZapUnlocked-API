---
title: 'Bloquear Contato'
icon: 'user-lock'
api: "POST /settings/block"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Bloqueia ou desbloqueia um contato do WhatsApp.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do contato com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="action" type="string" required>
  `"block"` para bloquear, `"unblock"` para desbloquear.
</ParamField>

<ResponseExample>

```json 200 Bloqueado
{
  "success": true,
  "message": "User 5511999999999 blocked successfully."
}
```

```json 200 Desbloqueado
{
  "success": true,
  "message": "User 5511999999999 unblocked successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

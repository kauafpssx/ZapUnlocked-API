---
title: "Botão Ligação"
icon: "phone"
api: "POST /messages/send-button-call"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma mensagem com botão de ligação. O usuário pode clicar para iniciar uma chamada de voz para o número especificado.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="callPhone" type="string" required>
  Número de telefone que será chamado ao clicar no botão.
</ParamField>

<ParamField body="button_text" type="string" required>
  Texto exibido no botão.
</ParamField>

<ParamField body="title" type="string" required>
  Título exibido no topo da mensagem.
</ParamField>

<ParamField body="text" type="string" required>
  Texto da mensagem exibido acima do botão.
</ParamField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Button sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

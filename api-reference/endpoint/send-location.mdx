---
title: "Enviar Localização"
icon: "location-dot"
api: "POST /send_location"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma localização geográfica para um contato ou grupo.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="lat" type="number" required>
  Latitude da localização (ex: `-23.5505`).
</ParamField>

<ParamField body="lng" type="number" required>
  Longitude da localização (ex: `-46.6333`).
</ParamField>

<ParamField body="name" type="string">
  Nome da localização exibido na mensagem (ex: `"Rua Principal, Centro"`).
</ParamField>

<ParamField body="address" type="string">
  Endereço completo (ex: `"Rua Principal, 1578 - Centro"`).
</ParamField>


<ResponseField name="success" type="boolean">
  Indica se a localização foi enviada com sucesso.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Location sent."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

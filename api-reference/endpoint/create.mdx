<ParamField header="x-api-key" type="string" required placeholder="SUA_SENHA_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma mensagem de texto simples para um número específico do WhatsApp.

### Placeholders Dinâmicos

Você pode usar variáveis no texto da mensagem que serão substituídas automaticamente:

| Placeholder | Descrição                                   |
| :---------- | :------------------------------------------ |
| `{{day}}`   | Dia atual (ex: 25)                          |
| `{{mon}}`   | Mês atual (ex: 12)                          |
| `{{yea}}`   | Ano atual (ex: 2024)                        |
| `{{hou}}`   | Hora atual (ex: 14)                         |
| `{{min}}`   | Minuto atual (ex: 30)                       |
| `{{sec}}`   | Segundo atual (ex: 45)                      |
| `{{phone}}` | Número do telefone do destinatário          |
| `{{name}}`  | _Nome do contato (se disponível na agenda)_ |

### Corpo da Requisição (Body)

<ParamField body="phone" type="string" required>
  O número de telefone do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="message" type="string" required>
  O conteúdo em texto da mensagem.
</ParamField>

<ParamField body="reply" type="string">
  Responde a uma mensagem específica. Pode ser o **ID da mensagem** ou o **texto
  exato** da mensagem (dependendo do parâmetro `type`).
</ParamField>

<ParamField body="type" type="string" default="id">
  Define como o parâmetro `reply` será interpretado: - `id`: Busca a mensagem
  pelo ID único do WhatsApp - `text`: Busca a mensagem mais recente com o texto
  exato fornecido (case-insensitive, normalizado)
</ParamField>

<Note>
  **Modo Text**: Quando `type: "text"`, a API busca no histórico local (últimas
  100 mensagens) a mensagem mais recente que contenha exatamente o texto
  fornecido em `reply`. Útil quando você não tem o ID da mensagem.
</Note>


<ResponseField name="success" type="boolean">
  Indica se o comando foi aceito pelo servidor.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva do status.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Mensagem enviada ✅"
}
```

```json 401 Erro
{
  "error": "Unauthorized",
  "message": "API Key inválida ou ausente"
}
```

</ResponseExample>

---
title: "Gerar Imagem IA"
icon: "image"
api: "POST /ai/imagine"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Gera uma imagem a partir de um prompt de texto usando o Meta AI integrado ao WhatsApp.

### Corpo da Requisição

<ParamField body="prompt" type="string" required>
  Descrição textual da imagem a ser gerada (prompt).
</ParamField>

<ParamField body="timeout" type="number" default="60">
  Tempo máximo em segundos para aguardar a geração da imagem.
</ParamField>

### Resposta

<ResponseField name="text" type="string">
  Texto descritivo retornado junto com a imagem.
</ResponseField>

<ResponseField name="hasImage" type="boolean">
  Indica se a imagem foi gerada com sucesso.
</ResponseField>

<ResponseField name="messageId" type="string">
  ID da mensagem com a imagem no WhatsApp.
</ResponseField>

<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "text": "Imagem gerada: um pôr do sol na praia...",
  "hasImage": true,
  "messageId": "true_5511999999999@c.us_ABCD1234"
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

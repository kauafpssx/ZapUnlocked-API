---
title: "Enviar Imagem"
icon: "image"
api: "POST /send_image"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Envia uma imagem a partir de uma URL pública.

### Corpo da Requisição

<ParamField body="phone" type="string" required>
  Número do destinatário com código do país (ex: `5511999999999`).
</ParamField>

<ParamField body="image_url" type="string" required>
  URL pública da imagem (jpg, png, etc).
</ParamField>

<ParamField body="caption" type="string">
  Legenda opcional para a imagem.
</ParamField>

<ParamField body="asDocument" type="boolean" default="false">
  Se `true`, envia a imagem como arquivo/documento (não inline).
</ParamField>

<ParamField body="fileName" type="string">
  Nome do arquivo exibido no WhatsApp (usado com `asDocument: true`, ex: `"foto.jpg"`).
</ParamField>

<ParamField body="reply" type="string">
  ID ou texto da mensagem para responder.
</ParamField>

<ParamField body="type" type="string" default="id">
  Como o campo `reply` é interpretado: `id` (padrão) ou `text`.
</ParamField>

<ParamField body="quoted_id" type="string">
  ID da mensagem a citar diretamente.
</ParamField>


<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Image sent successfully."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>

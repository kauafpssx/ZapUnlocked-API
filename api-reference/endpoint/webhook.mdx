Webhooks na **ZapUnlocked-API** são **stateless**. As instruções de para onde enviar os dados e o que enviar ficam embutidas no botão que o usuário clica.

<Note>
  **Vantagem**: Não é necessário banco de dados local para "lembrar" o que um botão deve fazer. O bot é completamente autocontido.
</Note>

## <Icon icon="gears" iconType="sharp-regular" color="#9d4fe6" size={22} /> Como Funciona

Ao criar um botão interativo, você define um objeto `webhook`. A API gera um token criptografado contendo essas instruções e o define como ID do botão.

Quando o usuário clica, a **ZapUnlocked-API**:

1. Recebe o clique.
2. Decifra as instruções do ID.
3. Substitui os **Placeholders** (veja abaixo).
4. Dispara o POST para sua URL.

### <Icon icon="brackets-curly" iconType="sharp-regular" color="#9d4fe6" size={22} /> Placeholders Disponíveis

Você pode usar variáveis no `body` ou `headers` do webhook que serão substituídas no momento do envio do evento:

| Placeholder | Valor |
| :---------- | :---- |
| `{{from}}` | Número do remetente |
| `{{text}}` | Texto da mensagem |
| `{{phone}}` | Mesmo que `{{from}}` |
| `{{id}}` | ID da mensagem |
| `{{requested}}` | Quantidade solicitada (fetchMessages) |
| `{{found}}` | Quantidade encontrada (fetchMessages) |
| `{{timestamp}}` | Timestamp UTC atual |

### <Icon icon="code" iconType="sharp-regular" color="#9d4fe6" size={22} /> Exemplo de Payload Recebido

Seu servidor receberá um JSON com a estrutura que você definiu, com os placeholders resolvidos:

```json
{
  "event": "button_click",
  "user": "5511999999999",
  "button": "Confirmar Compra",
  "time": "2026-01-13T14:50:00Z"
}
```

<Info>
  **Interatividade Stateless**: Perceba que o objeto de clique contém o ID selecionado. Esse ID foi decifrado pelo bot no momento do clique, permitindo que seu sistema identifique a ação sem um banco de dados local para "lembrar" o que aquele botão fazia.
</Info>

## <Icon icon="shield-check" iconType="sharp-regular" color="#9d4fe6" size={22} /> Segurança

Recomendamos validar a origem das requisições no seu servidor de webhook para garantir que venham apenas da sua instância da **ZapUnlocked-API**.

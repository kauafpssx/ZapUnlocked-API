---
title: "Desconectar"
icon: "right-from-bracket"
api: "POST /session/logout"
---

<ParamField header="x-api-key" type="string" required placeholder="SUA_CHAVE_SECRETA">
  Chave de autenticação da API.
</ParamField>

Desconecta imediatamente a sessão atual do WhatsApp e deleta os arquivos de credenciais locais (`creds.json`). Útil para trocar de conta ou forçar um novo QR Code.

<Warning>
  Esta ação é irreversível e exige um novo escaneamento de QR Code para restaurar o serviço.
</Warning>


<ResponseExample>

```json 200 Sucesso
{
  "success": true,
  "message": "Disconnected successfully (data cleared)."
}
```

```json 401 Não autorizado
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key. Send via x-api-key header (preferred) or ?API_KEY= query param."
}
```

</ResponseExample>
